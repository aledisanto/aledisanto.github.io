# Progetto di Reti di Calcolatori: Storia dell'Aston Martin

Sito web statico multipagina dedicato alla storia, ai modelli iconici (come la DB4 e la DB11) e all'evoluzione del celebre marchio automobilistico **Aston Martin**. 

Il progetto è stato sviluppato come elaborato pratico per l'esame di *Reti di Calcolatori* presso l'**Università degli Studi di Trieste**.

## Progetto Live
Il sito è configurato e ospitato pubblicamente tramite il servizio di cloud hosting di **GitHub Pages** al seguente indirizzo:
[https://aledisanto.github.io](https://aledisanto.github.io)

## Tecnologie e Competenze Sviluppate
* **HTML:** Strutturazione semantica dei contenuti su più pagine interconnesse (`index.html`, `pagina1.html`, ecc.).
* **CSS:** Gestione del layout, dello stile visivo e della formattazione del testo tramite un foglio di stile centralizzato (`style.css`).
* **Web Hosting & Deployment:** Configurazione e gestione del deployment automatico dell'applicazione web statica sfruttando l'infrastruttura di rete di GitHub.
* **Gestione Risorse:** Ottimizzazione e inclusione di risorse multimediali leggeri (`.png`) per garantire un caricamento rapido delle pagine lato client, riducendo la latenza di trasferimento dei pacchetti dati.

## Struttura della Repository
* `index.html` — Home page principale del sito web.
* `pagina1.html`, `pagina2.html`, `pagina3.html` — Pagine di approfondimento storico e tecnico sui modelli della scuderia.
* `style.css` — Foglio di stile CSS per il design grafico e il layout del sito.
* `aston007.png`, `astonDB4.png`, `astonDB11.png`, `astonSede.png` — Immagini utilizzati all'interno delle pagine web.
